# kadaiコマンド
![test]https://github.com/YS038/robosys2025/actions/workflows/test.yml
これは、文字を調べるコマンドです
cat filename | ./kadai keyword
filenameに調べるファイル名、keywordに調べる言葉を入れてください
OS:linux
python version：Python 3.12.3
